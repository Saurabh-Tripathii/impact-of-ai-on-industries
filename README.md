## Key Highlights
- Studied industry data to measure AI's impact.
- Used Likelihood Ratio Test to choose best predictive features.
- Applied Ordinal Forest technique for behavior analysis.
- Achieved:
  - Efficiency model accuracy: 91.50% (train) / 67.50% (test)
  - Expenditure model accuracy: 82.90% (train) / 60.34% (test)
  - Customer satisfaction accuracy: 77.73% (train) / 61.11% (test)
